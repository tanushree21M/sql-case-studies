# SQL Case Studies — Real Business Problems

10 real-world SQL problems solved with production-quality queries.

## Case Studies Covered

| # | Problem | Concepts Used |
|---|---------|--------------|
| 1 | Second Highest Salary | Subquery, DENSE_RANK |
| 2 | Customers With No Orders | LEFT JOIN, NULL filter |
| 3 | Top 3 Per Department | PARTITION BY, DENSE_RANK |
| 4 | Month Over Month Growth | LAG, CTE, Date functions |
| 5 | Running Total | SUM OVER, Window Functions |
| 6 | Find & Delete Duplicates | GROUP BY, HAVING, DELETE |
| 7 | Employee Manager Hierarchy | Self JOIN |
| 8 | Products Never Ordered | LEFT JOIN + NULL |
| 9 | Salary vs Department Average | Multiple Window Functions |
| 10 | Cohort Analysis | CTE, MIN, Date grouping |

## SQL Concepts Demonstrated

- Window Functions — ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD
- CTEs — Common Table Expressions
- Self JOIN
- Subqueries — correlated and non-correlated
- GROUP BY + HAVING
- CASE WHEN
- Date functions
- Data deduplication

## Who This Is For

These queries are commonly asked in Data Engineer and Data Analyst interviews at product companies like Razorpay, PhonePe, Swiggy, Zomato, CRED, Groww, and Meesho.
